### Image Rendering With a Downloadable PDF CTA

![Editorial healthcare still life showing an article image block and a download button for a publishing smoke test.](~/assets/images/uploads/smoke-test-image-rendering-with-a-downloadable-pdf-cta/smoke-image-pdf-rendering-hero-req.png)

This short article exists to verify one thing clearly: the current publishing path can render an article image and a visible PDF call to action on the same published page.

### What This Page Should Prove

The page should show a real image artifact supplied through the accepted publisher contract. It should also include a dedicated PDF download action that uses the exact PDF artifact generated for this workflow.

### Why This Matters

A payload can be accepted while a page still fails to render the media or download element. This smoke test checks the rendered outcome after publish, so the result is tied to what readers can actually see.
