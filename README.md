# Dr. Lurié Blog

Official source for the Dr. Lurié website and educational blog.

## Purpose

This site publishes science-led educational content on aging skin changes, age-related scent shifts, and future age-aware skincare concepts.

## Tech Stack

- Astro
- Tailwind CSS
- Markdown/MDX content

## Local Development

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:4321`.

## Build and Validation

```bash
npm run build
npm run check
```

## Content and Media

Project media files are hosted in a dedicated Netlify/GitHub media repository:

- Base URL: `https://kugelmedia.netlify.app/drlurieblog/`
- Repository path concept: `kugelmedia/drlurieblog/`
- Example file: `https://kugelmedia.netlify.app/drlurieblog/dr-lurie-portrait.jpg`

When adding new images, prefer that base URL unless the file already exists locally in this repository.
